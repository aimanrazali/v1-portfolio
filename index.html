Dental Management System

Homepage

Dashboard - Admin

Dashboard - Dentist

For my final year project, I developed a Dental Management System website specifically designed for the Kulliyyah of Dentistry at IIUM Kuantan. This website caters to dentists who specialize in treating autistic children. The project incorporates the use of PECS (Picture Exchange Communication System) to assist the children. The website features user login, dentist login, and admin login functionalities. Parents of the child can log in to access and review the treatment records and updates provided by the dentist. Dentists have the ability to log in, enter patient details, update treatment information, and manage appointment dates. The admin has comprehensive monitoring capabilities and the authority to add new dentists.
